# PUK Core API

Pardus Uyumluluk Kataloğu Merkezi Veritabanı ve Karar Destek Arayüzü.

## Kurulum ve Çalıştırma

1. Gerekli kütüphaneleri kurmak için:
```bash
task setup
```

2. Geliştirme sunucusunu başlatmak için:
```bash
task dev
```

API dokümantasyonuna erişmek için sunucu çalışırken `http://localhost:8000/docs` adresine gidin.
